## About

Grocery Guru AI is a modern web application built with React, TypeScript, Vite, Tailwind CSS, and shadcn-ui. The project focuses on creating a clean, responsive user experience while maintaining a simple and scalable codebase.

This repository demonstrates:

Component-based UI development with React

Type-safe development using TypeScript

Rapid iteration with Vite

Modern styling with Tailwind CSS and shadcn-ui

Deployment of a live production application
